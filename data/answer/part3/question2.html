Kiểm tra số hoàn hảo

Số hoàn hảo là gì?

Số hoàn hảo là một số nguyên dương mà tổng các ước nguyên dương thực sự của nó (các số nguyên dương bị nó chia hết ngoại trừ nó) bằng chính nó. 

Source
#include <stdio.h>

int main() {
  int n, sum = 0;

  printf("Nhập một số nguyên: ");
  scanf("%d", &n);

  if (n <= 0) {
      printf("%d không phải là số hoàn hảo.\n", n);
  } else {
      for (int i = 1; i <= n / 2; i++) {
          if (n % i == 0) {
              sum += i;
          }
      }
  
      if (sum == n) {
          printf("%d là số hoàn hảo.\n", n);
      } else {
          printf("%d không phải là số hoàn hảo.\n", n);
      }
  }

  return 0;
}
Input
Nhập một số nguyên: 6
Output
6 là số hoàn hảo.

Tại sao chỉ cần kiểm tra đến n / 2?

Một nửa của n (tức n / 2) là ước số lớn nhất có thể của n ngoại trừ chính nó.
