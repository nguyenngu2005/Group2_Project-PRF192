Vẽ hình thoi có 2 đường chéo bằng nhau
Source
#include <stdio.h>
#include <math.h>

int main() {
  int d, i, j;
  printf("Nhap do dai duong cheo hinh thoi:");
  scanf("%d", &d);

  if(d % 2 == 0){
      printf("Do dai duong cheo la so chan.");
      return 0;
  }

  int mid = d / 2;

  for(int i = 0; i < d; i++){
      for(int j = 0; j < d; j++){
          if(j == abs(mid - i) || j == mid + i ||j == mid + abs(d - 1 - i)){
              printf("*");
          }else{
              printf(" ");
          }
      }
      printf("\n");
  }
  return 0;
}

Giải thích các phép so sánh trong điều kiện

  • j == abs(mid - i): Điều kiện này kiểm tra xem vị trí cột j có trùng với 2 cạnh của nửa trái hình thoi hay không.
  • j == mid + i: Điều kiện này kiểm tra xem vị trí cột j có trùng với cạnh trên của nửa phải hình thoi hay không.
  • j == mid + abs(d - 1 - i): Điều kiện này kiểm tra xem vị trí cột j có trùng với cạnh dưới của nửa phải hình thoi hay không.
