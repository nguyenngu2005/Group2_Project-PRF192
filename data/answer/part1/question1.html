Vẽ tam giác cân

Cách 1

Để vẽ một tam giác cân, cách đầu tiên cũng là cách đơn giản nhất là chúng ta sẽ chạy 2 vòng lặp for lồng vào nhau như đoạn code bên dưới. 

Source
#include <stdio.h>
int main() {
  int h, i, j;
  printf("Nhap chieu cao cua tam giac: ");
  scanf("%d", &h);
  for (i = 0; i < h; i++) {
      for (j = 0; j < 2 * h - 1; j++) {
          if (j == h - 1 - i || j == h - 1 + i || i == h - 1) {
              printf("*");
          } else {
              printf(" ");
          }
      }
      printf("\n");
  }
  return 0;
}

/*Nếu bạn muốn hình tam giác có ruột (đầy sao),
  hãy thay điều kiện trong if bằng:
  (j >= h - 1 - i && j <= h - 1 + i) 
*/

Giải thích:

1. Dòng đầu tiên: Chỉ in một dấu * ở vị trí giữa dòng (ở cột h).
2. Dòng thứ hai trở đi: Các dấu * xuất hiện ngày càng xa vị trí giữa, tạo thành cạnh của tam giác.
3. Dòng cuối cùng: In một hàng đầy đủ 2 * h - 1 dấu * tạo thành đáy tam giác.

Và khi nhập input là 5 chúng ta sẽ có output như bên dưới:

Output
     *
    * *
   *   *
  *     *
 *********

Cách 2

Với cách 2 này chúng ta sẽ nâng cao hơn một chút, chúng ta sẽ chỉ chạy 1 vòng lặp và dùng 1 biến i duy nhất
Tuy nhiên đoạn code này sẽ ít trực quan hơn cách 1 và độ phức tạp vẫn sẽ à O(h2) 

Source
#include <stdio.h>
int main(){
  int h;
  printf("Nhap chieu cao cua tam giac: ");
  scanf("%d", &h);
  for(int i = 0; i < h * (2*h - 1); i++){
      if(i % (2*h - 1) == h - 1 - (i / (2*h - 1)) 
      || i % (2*h - 1) == h - 1 + (i / (2*h - 1)) 
      || i / (2*h - 1) == h - 1){
          printf("*");
      }else if(i % (2*h - 1) == 2*h - 2){
          printf("\n");
      }else{
          printf(" ");
      }
  }
  return 0;
}

/*Nếu bạn muốn hình tam giác có ruột (đầy sao),
  hãy thay điều kiện trong if bằng:
  (i % (2*h - 1) >= h - 1 - (i / (2*h - 1)) 
    && i % (2*h - 1) <= h - 1 + (i / (2*h - 1)))
*/

Giải thích:

1. Về phần điều kiện dừng của i, do chỉ chạy 1 biến duy nhất nên chúng ta sẽ cho i chạy từ 0 đến bé hơn h * (2*h - 1), do chiều cao của tam giác là h và đáy là 2*h - 1.

2. Có 2 phép tính cần lưu ý ở đây là i % (2*h - 1) và i / (2*h - 1):

  • i % (2*h - 1): Đây là cách lấy số cột hiện tại. Ví dụ khi h = 5 và i = 3 thì khi đưa vào công thức chúng ta sẽ có kết quả là 3. Ví dụ khác cũng với h = 5 nhưng i = 14, chúng ta sẽ có kết quả là 3.
  • i / (2*h - 1): Đây là cách lấy số dòng hiện tại. Bạn có thể lấy ví dụ h và i như trên để hiểu rõ.

3. Bây giờ khi đã hiểu 2 phép tính ở trên, bạn có thể hiểu nó là col (i % (2*h - 1)) và row (i / (2*h - 1)) tương ứng với i và j ở cách 1.

4. Về phần khi dán "\n" khi col đã đến cuối dòng, nghĩ là col == 2*h - 2, do mỗi dòng chỉ chạy đến bé hơn 2*h - 1.

5. Các trường hợp còn lại chúng ta sẽ in ra " ".
